# Vue 3 PeerJS

這是一個使用 Vue 3、PeerJS 和 TypeScript 構建的多功能通訊應用。本專案展示了如何在 Vue 3 應用中實現點對點 (P2P) 視訊通話和即時聊天功能。

## 功能

### 1對1視訊通話

- 高品質視訊通話

- 音訊/視訊控制 (靜音、關閉視訊)

- 接聽/拒絕來電

- 自動超時結束未接通的通話

### 1對1即時聊天室

- 文字消息交換

- 圖片分享功能

- 聊天邀請的接受/拒絕功能

- 聊天超時自動結束

## 技術棧

- Vue 3
- TypeScript
- PeerJS (用於 WebRTC 連接)
- Vite (構建工具)
